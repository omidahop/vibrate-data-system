- 🔐 سیستم احراز هویت کامل

- 📈 نمودارهای تعاملی
- 🔍 آنالیز افزایش‌های غیرعادی
- 📱 طراحی واکنش‌گرا
- 🎨 تم‌های روشن و تیره
- 👥 مدیریت کاربران
- 📤 خروجی و وارد کردن داده‌ها
